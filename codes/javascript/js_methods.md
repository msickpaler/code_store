## find

examples

```javascript
const array1 = [5, 12, 8, 130, 44];

const found = array1.find((element) => element > 10);
console.log(found); // expected output: 12

const found2 = array1.find((element) => element > 1000);
console.log(found2); // expected output: undefined
```
